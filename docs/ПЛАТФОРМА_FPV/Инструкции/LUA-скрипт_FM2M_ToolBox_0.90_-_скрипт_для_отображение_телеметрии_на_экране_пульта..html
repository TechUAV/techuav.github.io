LUA-скрипт FM2M ToolBox 0.90
Скрипт, для отображения телеметрии на экране аппаратуры управления, для монохромных экранов.
На экране отображается:
- Тип и уровень заряда АКБ.
- Напряжение на одно ячейке АКБ.
- Состояние "ARM".
- Режим полета.
- Дата и время (если настроены в пульте).
- Качество связи.
Скачать LUA-скрипт FM2M ToolBox 0.90
Нужно скачать архив на компьютер и распаковать. Содержимое папки FM2M_ToolBox_090 скопировать в
корень SD-карты пульта.
Запуск скрипта.
Скрипт запускается из меню SYS, страница TOOLS 1/7 - название скрипта FM2M_ToolBox_090.
Настройка скрипта.
Перед использование скрипт необходимо настроить.
1. Н странице главной скрипта необходимо выбрать пункт Config.

2. На второй странице выбрать протокол управления и версию прошивки.



3. На третей странице настроить тумблеры и их положение для “ARM” и режимов полетов.


В разделе ARM:
- выбрать строку SW name и совершить движение тумблером “ARM”,
- в строке SW pos указать положение тумблера при “ARM”.
В разделе Flight Mode:
- выбрать строку SW name и совершить движение тумблером, на котором настроено переключение
режимов полета,
- в строках SW pos Acro, SW pos Horizon, SW pos Angle указать, в каком положении тумблера
включаются эти режимы полета.
4. Вернуться на главную страницу и запустить программу по строке Dashboard.
