РЕЧЬ ПРО КРЫЛЬЯ (SC 350, ZALA, Орлан и т.п) 1) Необходимо позаботиться о наличии постоянного источника электричества от промышленной сети, которая будет питать всё необходимое оборудование (если запитка от промышленной сети возможна);
2)
Источник бесперебойного питания (бесперебойник)
,
который будет питать ваше оборудование на тот случай, когда промышленная сеть перестанет функционировать, а резервный источник питания ещё не заработал. 3) Резервный источник питания. Резервным источником питания является генератор. Лучше брать генератор мощностью от 7,5 кВт и так же иметь дополнительно к дизелю автономную электростанцию 2х150.000 мА*ч; 4) • 5 патч-кордов Ethernet 0,5 м; • 5 патч-кордов Ethernet 1 м; • 5 патч-кордов Ethernet 3 м; • 2 катушки патч-корда Ethernet 50 м; • кабель Ethernet длиной 200 м. Кабель 200 метров предназначен для возникающих потребностей, если какой-либо из других кабелей выйдет из строя, воспользовавшись «обжимкой» и «коннекторами rj-45» можно восстановить вышедший из строй патч-корд. Катушка 50м служит для создания «выноса» от антенны. 5) Клещи обжимные для RJ-45, служат для обжимки провода Ethernet, для создания патч-кордов; 6) 1-2 монитора с частотой от 144 Гц, предназначены для выведения картинки с ноутбука на большой экран. Упрощает работу при ведении разведки. 7) 2 шт SWITCH-POE устройство, которое подаёт питание на сети через Ethernet, по которому передаются данные. для объединения всех ноутбуков и устройств в одну локальную сеть. 8) Катушка питания 25 метров, 50 метров; 9) 2 шт. сетевого фильтра; 10) Запасной ноутбук с необходимым для работы программным обеспечением; 11) Запасной джойстик, который необходим для работы на вашем БпЛА. 12) Наземная метеостанция, которая для определения погодных условий в районе дислокации. На тот случай, когда скорость ветра не очевидна. 13) 3-4 маломощные радиостанции, для резервной связи внутри расчёта или при передвижениях на автомобилях. 14) герметичная плёнка, чтобы замотать розетки на случай осадков, зимний период времени; 15) Инструмент: Шуруповерт с комплектом бит, лопаты штыковая, молоток, строительный фен, топор, пила, бензопила, набор инструментов для автомобиля 128 предметов, набор отверток и бит для точных, гвозди, саморезы. 16) набор скалолаза (доп страховка, когда необходимо залезть куда-либо); 17) Большое количество стяжек большой длины; 18) Армированный скотч для сантехники 5 единиц; 19) COSMO FEN CA 12 2 шт; 20) эпоксидная смола, для ремонта БпЛА 21) соединитель Ethernet, для того, чтобы не падала скорость передачи данных про «разносе» антенны от НПДУ. 22) Баллон со сжатым воздухом, для чистки от пыли ноутбуков, продувки силовой установки, и очистки силовых от грязи, когда осуществляется посадка по самолетному; 23) кисточки для малярной работы, небольших размеров; 24) набор термоусадок разной длины и диаметра; 25) Тестер напряжения; 26) WD-40 для пневмы ZALA, SUPERCAM;
