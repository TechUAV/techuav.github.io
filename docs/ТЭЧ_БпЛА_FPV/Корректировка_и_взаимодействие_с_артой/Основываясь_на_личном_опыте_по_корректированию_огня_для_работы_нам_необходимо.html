Из темы Корректировка и взаимодействие с артой
Из источника ТЭЧ БпЛА | FPV
Основываясь на личном опыте по корректированию огня, для работы нам необходимо: 1) Наладить взаимодействие с арт.подразделениями на вашем участке фронта; 2) Телефон, желательно планшет с установленными на него программами (на выбор): - ZOV карты; - OfflineMaps; - AlpineQuest; Так же обязательно иметь на телефоне программу, которая предназначена для корректирования огневых средств. Примерами таких программ могут быть: - SPOTTER; - Глаз/Гроза; - Трепет; Телефоны и планшеты должны быть нулёвые, которыми не пользовались ранее в личных интересах. 3) Второй телефон, либо планшет по которому будет устанавливаться связь с артиллерийскими подразделениями, через защищенные и не защищенные каналы связи (Telegram и т.п). Возможен тот случай, когда будет использоваться «желтый телефон» - это вариант не столь удобный, связано это с тем, что при диктовке через телефон, абонент на другой линии может не расслышать цифры и начать просить продиктовать всё заново. Поэтому процесс ускорения передачи данных - это важный аспект сетецентрической войны. Так же не забываем, что нам всегда необходимо иметь резервный источник передачи данных, который будет работать на тот случай, когда наш основной способ связи вышел из строя; 4) иметь поблизости источник питания, который всегда будет питать ваши средства электроэнергией ( в том числе и при выходе промышленной сети из строя). 5) Ручка, блокнот - для оперативной записи поступающей информации; 6) Трансляция изображения, получаемого с БпЛА.
