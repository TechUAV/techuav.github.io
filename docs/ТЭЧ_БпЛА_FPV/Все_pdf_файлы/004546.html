Зміна потужності VTX в польоті
Аналогове відео видається в ефір в незашифрованому вигляді. І будь-який екіпаж фпв чи
підорський рер може перехопити ваші 2W відео з каміка від точки взлету, сидячи за 25км
від лбз, вичислити точку старту по орієнтирах і вбити. На бомберах ситуація ще гірше, так
як ви будете повертатися і спалите весь маршрут, по якому можуть повернути антену та
ще точніше вас спіймати.
Для запобігання цього є 4 методи - зміна потужності VTX в польоті, скрамбери, перехід на
кастомні відео частоти, перехід на цифрове відео.

Зміна потужності VTX в польоті - це найпростіше рішення, яке не вимагає додаткових
витрат, а тільки специфічної прошивки. В теорії відео на 25mW має бути видно на десяток
кілометрів, на практиці за 2-3км його вже ледве видно, тому цей метод достатньо
безпечний(і мастхев) і для каміків, і для бомберів.
Зміна потужності VTX 5.8 в польоті
Спочатку треба завантажити VTX table для свого VTX.
Ця вкладка буде заблокована, якщо в вкладці Ports не налаштовано протокол для VTX(в
стовбці Peripherals)


Тут налаштування аналогово VTX.
1) Виставлення відеоканалу
2) Pit mode та Low power disarm обов’язково відключаємо. Деякі VTX не виходять з піт
моду, або не перемикають потужність
3) Таблиця каналів для VTX
4) Завантаження таблиці каналів

Де знайти таблицію для свого VTX?
Гуглиш [назва свого vtx] VTX table. Знаходиш файл або json для нього. Вставляєш через ці
кнопки.
Наприклад, json для Akk dominator. Копіюєш його з цього сайту, а потім вставляєш через
кнопку Load from clipboard.

Після залиття таблиці треба обов’язково перевірити що:
1) Вибрано правильний протокол. Для цього перемикаємо відеочастоту, та дивимося що
вона реально змінюється. Якщо не змінюється - то скоріше за все або вибрано
неправильний протокол, або кабель від VTX до FC не підпаяно
2) Що потужність реально змінюється. Для цього перемикаємо потужність, та дивимося на
світлодіоди на VTX та показники PowerMeter

Далі переіменовуємо Label в цій таблиці, щоб на OSD не було видно реальну потужність.
1 рівень - V1 - мінімальна потужність. Зазвичай там від 25mW до 50mW
2 рівень - V2 - перша потужність після мінімальної. Зазвичай там від 200mW до 500mW.
3 рівень - V3 - максимальна потужність

Під час польоту оператор перемикає потужність по мірі погіршення відео. На підльоті до
ЛБЗ можна вмикати на максимум.

Далі встановлюємо перемикання цієї потужності з тумблера на пульті. Для цього треба
зайти на вкладку CLI та прописати такі команди:

vtx 0 3 0 0 1 900 1200
vtx 1 3 0 0 2 1300 1700
vtx 2 3 0 0 3(цю цифру треба змінити на максимальну з VTX table) 1800 2100
save

Пояснення команди: vtx <порядковий номер 0-1-2> <номер AUX поч з 0, у нас 3>
