Прошивка MILPACK v1.33
Руководство пользователя
Функциональность
Прошивка основана на ExpressLRS\Backpack 1.5.1 и добавляет следующие функции:
• Контроль двух VRX с разными связующими фразами (v1.33)
• Управление приемником FOXEER
• MILPACK_PHRASE - возможность задать альтернативную связующую фразу (v1.10)
• Поддержка X-диапазона
• TBS Fusion Control без внешнего рюкзака
Предполагается, что MILPACK используется совместно с MILERLS и модифицированной прошивкой для
VRX. Тем не менее, MILPACK также может работать со стандартной прошивкой ExpressLRS («VTX
Administrator»), за исключением некоторых функций (TBS Fusion).
Для успешной привязки MILPACK должен быть установлен на оба рюкзака, на TX и VRX.
Принцип работы
Модуль TX содержит отдельный подмодуль - Backpack, который может связываться с аналогичным
Backpack через WiFi
на VRX. Для этого рюкзаки необходимо привязать (иметь ту же фразу привязки или привязать вручную).
VTX Admin в скрипте LUA в пульте дистанционного управления отправляет команду switch через рюкзаки
в VRX.
Особенности крепления MILPACK
MILPACK не использует стандартную связующую
фразу. Привязка с пульта не работает.
Оба рюкзака должны быть установлены на одну и ту же MILPACK_PHRASE (по умолчанию - "milpack"). Эту
фразу можно изменить через веб-интерфейс (10.0.0.1)
Важный! MILPACK на VRX реагирует на команду с пульта дистанционного управления "Включить VRx
WiFi" в течение первых 60 секунд, даже если связующие фразы на рюкзаках не совпадают. Это дает
возможность проверить и изменить привязку на рюкзаке VRX. Пароль для WiFi сети - expresslrs
Распространение
Несите ответственность за распространение этой информации:
• Описание прошивки должно распространяться только в группах с верифицированными
участниками
• Передавайте скомпилированное встроенное ПО только тем, кого вы знаете или можете
проверить
• Направление заинтересованных лиц к разработчику (@olegator.40, @dankzb.03 в сигнале) для

получения новых версий и в качестве дополнительного способа проверки

Две связующие фразы
MILPACK позволяет установить две связующие фразы для управления двумя VRX. Это необходимо, в
первую очередь, для управления двумя TBS Fusion, либо связкой Skyzone + TBS Fusion.

Настройки SteadyView X

Краткий план действий

1. Обновите прошивку на SteadyView X для поддержки диапазона X.
2. Через USB прошити MILPACK в бекпек SteadyView X
3. Через WiFI прошити MILPACK в бекпек TX модуля
4. Убедитесь, что рюкзаки перепутаны и через VTX Admin с пульта дистанционного управления
можно переключить каналы на
SteadyView X
5. Обновление прошивки в модуле TX на MILELRS v2.56+
6. Настройте канал управления на пульте дистанционного управления, который будет отвечать
за коммутацию каналов
7. Настройка параметров VRX_CONTROL через веб-интерфейс модуля TX

Обновление прошивки VRX
1. При нажатой кнопке подключитесь к ПК через USB.
2. После этого устройство появится в проводнике, как если бы это была флешка.
3. Закиньте туда файл прошивки (например, SteadyView_X_LCD_DownTable_V2.bin), после того
как копирование закончится, можно отключать USB.
4. Далее включите VRX с помощью штатного гнезда и внешнего аккумулятора или вставьте его в
шлем.
5. Выполните сброс к заводским настройкам в меню.

Прошивка MILPACK в бекпек SteadyView X
1. Подключите внешнее питание к SteadyView X (не через USB)
2. SteadyView X меню -> Elrs -> ON, SteadyView X меню -> Elrs -> Upgrade
3. Подключите SteadyView X к компьютеру через USB
4. Запустити "Skyzone_Onboard_ESP32_Backpack\upload_skyzone_backpack.bat"
5. Следуйте инструкциям
6. Убедитесь, что прошивка успешно завершена
7. В случае ошибок переверните USB-кабель с другой стороны, попробуйте другой кабель.

Прошивка MILPACK в бекпек TX модуля
1. Пульт\Lua\ExpressLRS\WiFi подключение\Включить рюкзак Wifi
2. Если сеть "ExpressLRS TX Backpack" не появляется, необходимо проверить состояние
переключателей на модуле TX (пример для BetaFPV).
3. Перейдите в веб-интерфейс (10.0.0.1). Пароль от сети WiFi - expresslrs
4. Выберите файл "MILPACK\ESP_TX_Backpack\firmware.bin.gz" и обновите прошивку.

Проверка рюкзаков бинду
1. Пульт\Lua\ExpressLRS\VTX Admin\
2. Выберите новую полосу и канал
3. "Отправить VTx"
Если VRX не реагировал на команду, то и рюкзаки не сбивались. Рекомендуем активировать WiFi на
обоих рюкзаках и проверить привязку фразы в веб-интерфейсе.

Настройка TBS Fusion

Управлять TBS Fusion можно двумя способами: припаять внешний рюкзак ExpressLRS или настроить
МИЛПАК 1.20+. Последний метод работает только вместе с MILELRS v2.69+ или v3.10+

Краткий план действий

1. Обновление TBS Fusion
• Установка, запуск агента TBS
• Подключение TBS Fusion к компьютеру через USB
• Обновите прошивку до последней версии: TBS Agent>Fusion Module>Firmware>v2.38>Flash
• Включити на пристрої WiFi: Fusion MENU>CRSF>Fusion WiFi>General>WIFI AP>ENABLE
• Подключение к сети устройства (tbs_fusion_XXXXXXXX)
• Откройте веб-интерфейс http://192.168.4.1
• Обновите прошивку модуля WiFi: Upgrade> FUSION_0x000350xx_0x0219
• Активувати ESPNOW: Fusion MENU>CRSF>Fusion WiFi>PRO>ESP NOW>ENABLE
2. Настройка модульного рюкзака TC
• Включити WiFi на ТХ бекпак: LUA>ExpressELRS>WiFi Connectivity>Enable Backpack WiFi
• Откройте веб-интерфейс http://10.0.0.1 (Пароль для сети WiFi - expresslrs)
• Измените фразу привязки на имя TBS Fusion WiFi, например, tbs_fusion_A2B3D4F5E6
3. Протестируйте работу: LUA>VTX Administrator>Send VTx

TBS Fusion состоит из двух модулей - Fusion Module и WiFi модуля. Каждый модуль имеет отдельную
прошивку. Модуль Fusion должен быть обновлен через TBS Agent, модуль WiFi должен быть обновлен
через веб-страницу.

Прошивку для модуля WiFi можно найти в MILPACK или скачать самую свежую:

Настройка FOXEER

Foxeer позволяет переключаться между 8 выбранными каналами (Избранное) с помощью трех контактов.
Функционал добавлен благодаря @IVANHOE и @Ai Yk
Подробная инструкция находится в файле FoxeerVRX-MilPack.pdf

Краткий план действий

1. Прошите Foxeer_ESP8286_RX_Backpack в любой модуль Generic Esp8285 RX (список
совместимых ресиверов смотрите в инструкции к MILERLS). Рекомендуется – Happymodel
ES900 RX
2. Подключите три контакта модуля RX к плате преобразования логического уровня 3,3 В в 5 В
3. Выход конвертационной карты – до трех контактов на каждый ресивер Foxeer
4. Убедитесь, что MILPACK TX и VRX имеют одинаковые фразы привязки
5. Для переключения каналов используйте каналы А1... А8.
6. Если вы используете плату преобразования логического уровня, которая инвертирует сигнал,
используйте каналы B1... В8

Проблемы с обновлением Wi-Fi
Существует типичная проблема с WiFi-соединением, когда модули expresslrs обрываются через
несколько секунд после подключения.
Это известная проблема для Windows и более поздних версий. Причина в том, что Windows пытается
найти соединение, которое есть в Интернете.
Автор не знает надежных способов решения этой проблемы, если у вас есть такой опыт, отправьте
сообщение на почту.

Альтернативные методы:
1. Подключение через телефон Android. При подключении скажите «Всегда подключаться», когда
появится сообщение о том, что Интернет может быть недоступен. Рекомендуется отключить мобильную
передачу данных.
2. Windows: Отключите функцию "Подключаться автоматически" для основной сети, включите для
expresslrs
сети, ловите момент, пока устанавливается соединение.
3. Windows: используйте дополнительный адаптер Wi-Fi для подключения к сети expresslrs
4. Windows: подключение компьютера к Интернету через Ethernet
