Примитивный аналог COS для пультов RM700
(работает только с пультами модели RM700)


Описание:
Утилита представляет собой примитивный аналог русского COS и позволяет
работать с одного пульта RM700 c разными типами дронов DJI.

После установки этой утилиты на пульт RM700, он будет совмещать
возможности пультов: RC Pro(RM510), RC Enterprise(RM510b) и RC
Plus(RM700), а список поддерживаемых дронов будет расширен до тех
моделей, которые эти пульты поддерживают: Mavic3, Mavic3Classic,
Mavic3Pro, Mavic3Cine, 3T, 3E, 3M, Matrice30. Переключение режима
выполняется в один клик - по соответствующей иконке.

Данная утилита была скопирована с трофейного хохлопульта RM700 и
дополнена файлами для установки ее копии на такие же пульты.

Установка:
Данная утилита является аналогом COS и во избежание конфликтов не ставьте ее на пульты
поверх COS, так как функционал по своей сути один и тот же.

Проверьте перед установкой, что:
- устанавливаете именно на пульт RM700;
- ранее на пульте не был предустановлен COS;
- извлекли внешнюю SD-карточку (при наличии) из пульта;
- скопировали все нужные данные с пульта.


Все необходимые файлы имеются в архиве с программой. Разархивируйте ее,
а затем запустив CMD.EXE и перейдите в каталог с ней в командной строке.
Все команды в дальнейших примерах приведены относительно изначальной
структуры каталогов, поэтому не переименовывайте ничего.

1) Перед началом установки обязательно подключитесь к устройству и
проверьте результат вывода команд:
1
- adb.exe shell "getprop | grep ro.build.product" → rm700;
- adb.exe shell "getprop | grep ro.serialno" → 4LF…;
- adb.exe shell "file /data/original_version" → … cannot open;

Если какая-та из первых двух команд не совпала, то отложите пульт (это
может быть другая модель или ревизия). Установка пока что невозможна.
Если для третьей команды вывод отличен от указанного выше и содержит
фразу "ASCII text", то это означает, что ранее была использована данная
утилита.

Если для первой команды вывод отличен от "rm700", то это означает, что
ранее кто-то невнимательно читал данную инструкцию или там был
предустановлен COS.

2) Если все подготовительные шаги успешны, то можно запускать
автоматическую установку данной утилиты - скрипт installer.bat.
Длительность установки ~10 минут.

Установка скрипта выполняется в 10 этапов, при этом необходимо
подтверждение перехода к следующему этапу клавишей Enter.

В ходе работы установочный скрипт сделает следующие действия:
- повторно проверит текущую конфигурацию;
- осуществит сброс пульта в первоначальное состояние;
- зальет необходимые файлы прошивки, обновит ее;
- зальет дополнительные файлы, выдаст права;
- зальет необходимые apk, установит их, выдаст им права.




1
Программа adb.exe (Android Debug Bridge) может быть скачана с официального сайта
(https://developer.android.com/tools/releases/platform-tools) или запущена из локальной копии в каталоге
platform-tools-latest-windows.

Использование пульта:
После успешной установки на рабочем столе появится ряд приложений:

Данные приложения позволяют переключать пульт RM700 в режим работы
других пультов – для соответствующих дронов.

После запуска соответствующих приложений происходит переключение – на
нужную модель пульта:
Icon Application Description
- При нажатии загружает оригинальную
RC Plus прошивку (слот “a”);
- Активное приложение – DJI Pilot 2.

- При нажатии загружает обновленную
RC Pro прошивку (слот “b”);
- Активное приложение – DJI Fly.

- При нажатии загружает обновленную
RC Enterprise прошивку (слот “b”);
- Активное приложение – DJI Pilot 2.

- Автоматически запускается при старте со
Autostart слота “b”;
- Стартует приложение – DJI Pilot 2 / DJI Fly.


Отметим, что для успешной смены типа пульта реализован
полуавтоматический механизм подмены файлов, а именно: после нажатия
на одну из иконок (Plus, Pro, Enterprise) пульт перезагрузится, подменит
файлы и выключится. Далее необходимо вручную запустить пульт, а
приложение DJI стартанет уже автоматически.

Вопросы/Ответы:
- На каких версиях проверялась установка?
- Проверено на версиях 01.18.00.07 (01.02.0367), 01.27.00.15 (03.02.0907) и 01.33.00.13
(05.02.1106) для rm700.

- Можно ли обновлять прошивку после установки утилиты? Сохранится ли
функционал?
- Нет, нельзя. Функционал не сохраняется.

- Можно ли выключить gps на пульте?
- Да, использовать gps_OFF.bat.
