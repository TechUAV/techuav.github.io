Ремонт крепления подвеса Мавик 3
Было сломано крепление камеры, из-за чего камера не стабилизировалась и ее раскачивало.
Для ремонта использовались детали, распечатанные на 3Д-принтере:
- крепление подвеса,
- фиксатор для шлейфа 1,
- фиксатор для шлейфа 2.

Для замены пришлось разобрать почти ВЕСЬ дрон, сняли верхнюю крышку, плату GPS, плату.
При разборке нужно соблюдать осторожность – очень много тонких и маленьких шлейфов, которые можно
вырвать или порвать.
При сборке рекомендуем проклеить края шлейфов подключения камеры к материнской плате, тут подробно
показали как, см. на фото ниже.
Было установлено новое крепление, и собран дрон.
