Скачать прошивку ELRS
Для чего нужно скачивать прошивку с приемника/передатчика?
- Нужно прошить трофейный приемник/передатчик, а пароль от wi-fi не известен.
- Передать на анализ специалистам нестандартную прошивкt. Сейчас попадается много приёмников и
передатчиков, которых нет в обычном ExpressLRS конфигураторе.
Общее правило перед тем, как что-то делать – сделайте резервную копию прошивки. Для этого следует
подключиться по wi-fi и перейти по адресу http://10.0.0.1/firmware.bin Так же стоит сохранить себе
настройки, перейдя по адресу http://10.0.0.1/hardware.json, далее “сохранить как”.
Скачать прошивку с приемника/передатчика ERLS можно несколькими способами.
1. Программа esptool.
2. Программа read firmware (esptool easy).
3. Трофейная программа flash bin bfpassthrough.
Программа esptool/read firmware.
Скачать и распаковать архив с программой в любое удобное место на компьютере.
Для скачивания прошивки с приемника с помощью программы esptool или read firmware необходимо
дополнительное оборудование:
- TTL-адаптер, он нужен для того, чтобы подключить приемник к компьютеру (передатчик подключается
просто с помощью USB-кабеля).
Приемник нужно припаять к TTL-адаптеру (аналогично как к полетному контролеру):
- RX приемника – TX TTL,
- TX приемника – RX TTL,
- 5V приемника – 5V TTL,
- GND приемника – GND TTL.
Для скачивания прошивки с передатчика достаточно перевести его в режим загрузки (BOOT).
Например:
для BetaFPV 915 Micro TX нужно переключатели установить в положение:
1-2 и 5-7 – в нижнее положение,
3-4 – в верхнее положение,
а для HappyModel ES900TX, снять крышку и поставить переключатели (отсчет от USB-порта):
1-4 – в нижнее положение,
5-6 – в верхнее положение.
И подключить с помощью USB-кабеля.


BetaFPV 915 Micro TX HappyModel ES900TX


Порядок скачивания прошивки с помощью esptool.
1. Зажать кнопку BOOT на приемнике (где есть кнопка) или замкнуть контакты BOOT (где нет кнопки).
2. Подключить TTL-адаптер к компьютеру (приемник припаян, BOOT зажат).
3. В диспетчере устройств компьютера, в разделе порты (COM и LPT) посмотреть номер порта, по
которому подключился TTL-адаптер (на разных компьютерах он будет различаться).

4. Открыть папку с esptool.
5. Запускаем командную строку из этой папки, для этого в адресной строке пишем CMD и нажимаем
ENTER.


6. К командной строке вводим команду:
esptool.exe -p COMXX -b 460800 read_flash 0 ALL filename.bin
- вместо COMXX пишем номер вашего порта (который смотрели в пункте 3), например COM22,
- filename.bin тут указываем произвольное имя (главное расширение .BIN), с таким названием
выгрузиться файл прошивки,
нажимаем ENTER,

7. Если все было подключено правильно, то программа подключится к приемнику и пойдет процесс
скачивания прошивки, по окончании в папке с программой сохранится файл (с тем названием,
которое указали в команде).



Для передатчика все также, только начиная с пункта 3.

Порядок скачивания прошивки с помощью read firmware (облегченный вариант запуска esptool).
1. Зажать кнопку BOOT на приемнике (где есть кнопка) или замкнуть контакты BOOT (где нет кнопки).
2. Подключить TTL-адаптер к компьютеру (приемник припаян, BOOT зажат).
3. Открыть папку с read firmware.
4. Запустить файл READ.cmd.

5. Если подключение было выполнено правильно, программа подключиться к приемнику и сохранит
прошивку в файл my001.bin в папку с программой.




Для передатчика все также, только начиная с пункта 3.

Трофейная программа flash bin bfpassthrough (только для приемника).
Не сработала с полетным контроллером на прошивке Milbeta.

Данный метод более простой чем через esptool, не требует дополнительного оборудования.
Нужен только приемник и полетный контроллер.

Порядок скачивания прошивки с помощью flash bin bfpassthrough.
1. Скачать и распаковать архив с программой в любое удобное место на компьютере.
2. Подключить приемник к полетному контроллеру (пайкой или разъемом).
3. Подключить полетный контроллер к компьютеру с помощью USB-кабеля.
4. Открыть папку с flash bin bfpassthrough.
5. Запустить файл download_receiver_fw.cmd.

6. Программа должна подключиться к приемнику и запуститься процесс скачивания прошивки.

7. По окончанию, программа сохранит прошивку в папку с названием from_rx_firmware.bin.




Чтобы в файле прошивки найти пароль от Wi-Fi нужно:
1. Открыть файл с помощью блокнота.
2. Задать поиск по фразе: 10.0.0.1.
3. Рядом будет указаны пароль и название сети.

Для приемников на MilERLS чтобы посмотреть пароль Wi-Fi:
1. Открыть файл с помощью блокнота.
2. Задать поиск по фразе: encryption_key.
3. Рядом будет указан пароль.

Как подключиться по WI-Fi к приемнику на MILELRS
. ℹ️ MILELRS - прошивка оппонентов, для управления FPV. Не имеет стандартного пароля: каждая шифрованная пара (приемник-передатчик) имеет свой уникальный пароль.
К приемнику на MILERLS подключиться по Wi-Fi можно двумя способами
1. Получить пароль, скачав прошивку.
Понадобиться TTL адаптер и программа "
". Получать пароль необходимо в том случае, если нужно посмотреть настройки приемника (например частоты). В ином случае можно сразу прошивать на свою прошивку (п.2).
2. Сбросить настройки до значений «по-умолчанию».
Сброс настроек нужен для того, чтобы можно было прошить приемник на свою прошивку (например KapibaraLRS). - подать питание на приемник (через полетник, TTL); - как приемник запустится (начнет мигать светодиод) – зажать кнопку BOOT (где есть) или замкнуть контакты BOOT (где кнопки нет) не менее чем на 20 секунд; - после перезагрузить приемник; - название сети сбросится на ExpressRLS RX, и стандартный пароль expressrls.
3. Если на приемнике не работает кнопка BOOT.
Попробуйте аккуратно удалить кнопку с платы и замкнуть контакты вручную (с помощью проволоки или пинцета), далее см. п1. Если не помогло, то прошить с помощью полетного контроллера - подробнее в инструкции "
