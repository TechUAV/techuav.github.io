с помощью программы 1001_v36_M3.
Так как эта программа, по сути, копия прошивки 1001й (версии 36), то мы условно всегда будем называть ее
«1001_v36_M3».
Прошивка дронов DJI Mavic 3 на 1001ую прошивку версии 34 или 36 с помощью программы 1001_v36_M3.
! Только для DJI Mavic 3
Перед прошивкой на компьютер заранее необходимо установить дополнительные программы драйвера.
Установку программ и драйверов необходимо осуществлять только под учетной записью Администратора
компьютера.
Из папки “_Install” устанавливаются следующие программы и драйвера:
1. NET_SDK.exe - необходима для запуска программы.
2. DJI_Assistant_2.exe - она при установке настраивает драйверы для DUML коммуникации.
3. Ниже две программы устанавливаются только при наличие интернета (установить обе):
Python_3.cmd - для установки Python;
Python_zav.cmd - для установки зависимостей.
4. Системные файлы DLL_LibUSB.exe.
5. Драйверы LibUSB для работы с рекавери режимом Мавика:
Для установки драйверов LibUSB использовать программу Zadig (zadig-2.7.exe):
- выбрать Device - Load Preset,
- выбрать файл dji_recovery_zadiq_preset.cfg,
- выбрать драйвер libusb-win32 (v1.2.6.0),
- выбрать действие «Install driver»,
- нажать Install driver.
6. Установить драйвера ADB и Fastboot с помощью ADB_Fastboot.exe.
Программа 1001_v36_M3 не требует установки.
Отключаем интернет! Запускаем 1001_v36_M3.exe
После запуска программы:
1. Включить дрон и подключить его к компьютеру с помощью кабеля (проверить уровень заряда АКБ,
процесс прошивки занимает 10-15 минут).
2. В правом боковом меню перейти во вкладку “Team“.
3. Нажать кнопку “Update USB list“ и “Connect DUMPL“.
4. В правом верхнем углу надпись «DUMPL Disconnected» заменится на «DUMPL Connected» и поменяет
цвет с красного на зеленый.
5. Вернуться во вкладку “Run Programs“.
6. Выбрать версию прошивки (34 или 36, рекомендуем 36):
- кнопка “Flash 1001 v34” или “Flash 1001 v36”.
7. При прошивке первого дрона программа выдаст ошибку, что не может подключиться к устройству.
Необходимо будет доустановить драйвера.
- не отключая дрон от компьютера, открыть «Диспетчер устройств»;
- найти "Android" в «Other Devices»/ «Другие устройства»;
- нажать правой кнопкой мыши;
- выбрать “Обновить драйвер”;
- выбрать “Найти драйверы на этом компьютере”;
- выбрать “Выбрать драйвер из списка доступных драйверов на компьютере”;
- выбрать драйвер по цепочке: ADB Interface/ SAMSUNG Android ADB Interface Version: 5.14.0.0
[05.02.2010];
- согласиться и закрыть окно.
8. Отключить дрон от компьютера, перезагрузить и начать с пункта 1. Теперь ошибки в подключении не
будет и дрон успешно прошьется.
При последующих прошивках установка данных драйверов не потребуется.
