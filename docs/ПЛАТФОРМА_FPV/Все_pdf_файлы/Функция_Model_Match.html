Функция Model Match
Model Match - это функция, которая предотвращает случайный выбор неправильной модели в пульте и
управления с неожиданным пультом или конфигурацией ELRS.
Она позволяет пользователю подключать выбранную модель только к определённому приёмнику.
Аналог двухуровневой идентификации.
Например: в пульте настроены две модели для управление разными типами БВС (квадрокоптер, самолет), так
как в передатчик нельзя установить две разные бинд-фразы, то на помощь приходит функция Model Match.
Для каждой модели управления устанавливается свой идентификатор (для модели квадрокоптера – 01, для
модели самолета – 02), в приемниках также прописываются идентификаторы (в квадрокоптере – 01, в
самолете – 02). И при попытке сопряжения, даже если бинд-фраза одинаковая, но разные идентификаторы
Model Match – соединения не установится. В результате нельзя подключиться к квадрокоптеру с моделью
управления самолетом.
Настройка Model Match.
- Настройка пульта.
1. Включить пульт и перейти в раздел «Модели», кнопка MDL, вторая страница SETUP 2/12
2. Раздел Internal RF или External RF (в зависимости какой передатчик используется внутренний или внешний).
3. В строке Receiver выставить идентификатор, значения от 01 до 63.
4. Перейти в раздел «Система», кнопка SYS, строка ExpressLRS.
5. В строке Model Match установить значение On.
Также, в этой строке будет отображаться выбранный в модели идентификатор.
- Настройка приемника.
1. Включить приемник и перевести его в режим WiFi.
2. Подключиться по WiFi к приемнику и открыть веб-интерфейс.
3. На странице MODEL, прокрутить вниз, до раздела Model Match.
4. В разделе Model Match установить галку Enable Model Match, и в строке Model ID ввести значение
идентификатора, указанное в передатчике.
