�� СКАЧАТЬ ПРОШИВКУ MILBETA
Есть две версии прошивки, на базе Betaflight 4.3.3 и Betaflight 4.5.1 (выбираем под версию полетного
контроллера).
! Перед установкой прошивки ОБЯЗАТЕЛЬНО необходимо сделать дамп настроек.
Либо через раздел ПРЕДУСТАНОВКИ\Сохранить резервную копию или через командную строму - dump all
Прошивка устанавливается через Betaflight Configurator – обновить прошивку.
- выбрать – полное стирание чипа.

- нажать кнопку «Загрузить пошивку (local)), указать путь к папке 433 или 450 (в зависимости от выбранной
версии), выбрать прошивку под свой полетный контролер – файл с разрешением. HEX.
- нажать кнопку «Прошить прошивку».
После окончания процесса прошивки, для версии 4.3.3 необходимо загрузить файл конфигурации – папка
config, файл с расширением .config. Конфигурация заливается через командную стоку. Файл также
выбирается по модели полетного контролера.
- загрузить, ранее сохраненный дамп настроек.
*Если после прошивки MILBETA на базе 4.5.1 возникли проблемы с OSD, выполните команду в командной
строке: "set displayport_msp_fonts=0,0,0,0"
! Для корректного отображения OSD, также нужно загрузить шрифты.
- Открыть Менеджер шрифтов из закладки OSD
- Выберите файл milbeta.mcm, находиться в папке MILBETA
- Кнопка «Загрузить шрифт»
