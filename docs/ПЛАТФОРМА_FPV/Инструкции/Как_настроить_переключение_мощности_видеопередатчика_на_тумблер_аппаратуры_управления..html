Настройка переключения мощности видеопередатчика через тумблер на пульте.
1. Подключаем дрон, запускаем Betaflight Configurator.
2. Выбираем пункт – ВИДЕОПЕРЕДАТЧИК.
Необходимо запомнить в таблице мощности порядок, под каким номером, какая установлена
мощность.
В данном случае:
1 – 25 мВт
2 – 400 мВт
3 – 800 мВт
4 – 1 600 мВт
3. Дальше переходим в раздел Командная строка.

4. В командной строке необходимо использовать команду: vtx 0 0 0 0 0 900 900
0 0 0 0 0 900 900
номер номер сетка номер мощность минимальное максимальное
ячейки переключателя частот канала по значение значение
памяти (тумблера) сетке переключателя переключателя
- каждое положение переключателя нужно сохранять в свою ячейку памяти
- тумблеры номеруются с 0, а не с 1.


- ставится не сама мощность, а ее положение в сетке мощности


Положение переключателя

Переключатели на пульте 3-позиционные, фиксированные.
- 1-е положение – от 900 до 1100
- 2-е положение – от 1200 до 1600
- 3-е положение – от 1700 до 2100

5. Пример настройки на стоковом видепередатчике BLITZ VTX 1.6W.
Нужно: на переключатель С (AUX 4), настроить переключение на 1-е положение - 25 мВт, 2-е
положение – 800 мВт, 3-е положение – 1 600 мВт.
В командной строке нужно прописать следующие команды:
vtx 1 3 0 0 1 900 1100 – кнопка ENTER
vtx 2 3 0 0 3 1200 1600 – кнопка ENTER
vtx 3 3 0 0 4 1700 2100 – кнопка ENTER
SAVE – кнопка ENTER

ИТОГИ.

Данные настройки необходимы для обеспечения БЕЗОПАСНОСТИ оператора.
1. Взлетать нужно на малой мощности передатчика, так как канал видео НИКАК НЕ ШИФРУЕТСЯ.
Поэтому к нему может подключиться любой, у кого есть соответствующее оборудование.
Это сразу выдаст место взлета.
Малая мощность (25 мВт), не даст видеосигналу уйти дальше 100-300 метров.
2. Дальше, при ухудшении связи можно переключиться на большую мощность по мере удаления.
